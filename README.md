# La roue des conseils — Français A2

Application éducative interactive pour pratiquer l’expression orale et donner des conseils en français.

## Fonctionnalités

- dix situations affichées directement sur la roue ;
- rotation avec la souris, le tactile ou le clavier ;
- projection de la phrase sélectionnée ;
- élimination des phrases déjà travaillées ;
- restauration de toutes les situations ;
- rappel de quatre structures : `il faut`, `il vaut mieux`, `devoir` et l’impératif.

## Développement

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```
